MICROSOFT CORPORATION CREDITS
=============================

The TiledResources sample uses several assets that were generated using data
from NASA missions.  These assets include:
    diffuse.bin
    normal.bin
    geometry.vb.bin

Each of these assets was constructed using original data from:
    http://pds-geosciences.wustl.edu/missions/mgs/megdr.html
    http://www.mars.asu.edu/data/mdim_color/

The terms of use of this data can be found at:
    http://www.nasa.gov/audience/formedia/features/MP_Photo_Guidelines.html
    http://themis.asu.edu/terms
